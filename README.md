# dotfiles

My dotfiles.

## Features

### nvim

- [lazy.vim](https://github.com/folke/lazy.nvim) as plugin manager
- [treesitter](https://github.com/nvim-treesitter/nvim-treesitter)

#### Custom keymaps

| Keymap | Description |
|-|-|
| \<leader\>fp | copy current buffer's file path to clipboard |
